# Path of Light Yoga Studio – Chapter 7

This repository contains the Chapter 7 implementation of the **Path of Light Yoga Studio** website.  
The project focuses on enhancing the site’s layout and responsiveness using modern HTML and CSS techniques.

## Chapter Focus
Chapter 7 emphasizes:
- Semantic HTML structure
- External CSS styling
- Layout design using CSS (including Flexbox)
- Responsive design with media queries
- Improved visual consistency and usability

## Files Included
- **HTML files**  
  Contain the structured content for the Path of Light Yoga Studio website using semantic elements.

- **CSS file(s)**  
  External stylesheet(s) used to control layout, colors, typography, spacing, and responsiveness.

## Features
- Clean, semantic HTML structure
- External CSS for all styling
- Responsive layout that adapts to different screen sizes
- Styled navigation and consistent page layout
- Improved readability and user experience

## How to View
1. Download or clone the repository.
2. Open any HTML file in a web browser.
3. Resize the browser window to observe responsive behavior.
